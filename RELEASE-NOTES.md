# **The first version of Weather (v1.0.0).**
## Weather has one variation, Weathera (basically a weather alternative, that wait for the user press enter to exit).
### Type ```sudo ln -s ./weather.sh /usr/local/bin/weather && sudo ln -s ./weathera.sh /usr/local/bin/weathera && sudo chmod +x ./weather.sh ./weathera.sh``` to enable execute both apps and use both in any directory.
